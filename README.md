# Sellora AI

**Идея:** сервис для генерации продающих визуалов под маркетплейсы (Wildberries, Ozon, Яндекс.Маркет) — загрузка фото товара, получение консистентных снимков, слайдов карточки и промо-роликов.

Проект **в разработке**. Сейчас реализован интерфейс (визуал и сценарии), без подключённого бэкенда и реальной AI-генерации.

## Что есть сейчас

- **Лендинг** — главная страница.
- **Дашборд** — приветствие, блоки «токены», «генерации», «рефералы», быстрые действия (ссылки).
- **Страница генерации** — загрузка 1–4 фото, выбор режима (Фото / Полная карточка / Видео), категория, пожелания, кнопка «AI-идея» (случайная подсказка из списка), **демо-сценарий** «генерации» (таймер и плейсхолдеры результатов; реальная генерация пока не подключена).
- **Токены, рефералы, история, настройки** — страницы интерфейса.
- **Вход / регистрация** — экраны (логика авторизации не подключена).
- **Политика конфиденциальности, условия использования, оферта** — отдельные страницы.
- Адаптивная вёрстка, тёмная тема, shadcn/ui.

Почему стоит смотреть в сторону Sellora: открытый исходный код, прозрачная разработка, цель — удобный и доступный инструмент для селлеров; по мере готовности будут подключаться реальная генерация, баланс токенов и платежи.

## Быстрый старт

```bash
git clone https://github.com/ForseJDM/sellora-ai.git
cd sellora-ai
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000). Сборка: `npm run build` и `npm start`.
